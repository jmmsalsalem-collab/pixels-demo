# Pixels Smart Platform — Demo

AI-powered CRM, Task Management, Client Portal & Full Agent System.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import repo
3. Framework: Next.js (auto-detected)
4. Click Deploy — done.

## Local Dev

```
npm install
npm run dev
```

Open http://localhost:3000
